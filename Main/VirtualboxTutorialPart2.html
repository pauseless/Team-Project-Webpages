Snapshots, Installs and Pausing

Why do I need to know about this?

Snapshots

We all make mistakes, sometimes we install something wrong. Add a bit of code that breaks everything, or just brick the entire sytem with an update

Snapshots allow you to essentially rewind back to when you took them, finally making it possible to rewind back to before you installed that program that slowed down your entire PC!

More Installations

It can be important when handling different virtual machines to know how they handle with different filesystems. And if you have a lot to test, then you may want something like Unattended Install to speed up the process.

Never forget aswell that if you were testing what happens when you run malware, you'd want to be able to rollback to before it infected your system, and check how it handles multiple drives.
