Virtual Machines

Introduction

What is a virtual Machine?

Computer Virtualization

Virtual Machines are essentially a simulated computer that runs on a main computer.

So for example, Your Windows 11 machine can run a VM (Virtual Machine) of Linux or Mac OS or Windows XP all without having to make a new computer or install those onto your computer!

Use in Cyber Security

Virtual Machines get used within Cyber-Security of isolating environments, testing and simulating machines all while being contained. So if you load a virus on a virtual machine, in most circumstances only the virtual machine storage will be affected. You can also test how different machines react such as different windows versions or different operating systems entirely.

In this Module we'll be creating a isolated environment where we can test machines using Virtualbox
